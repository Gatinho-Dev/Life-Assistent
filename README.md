# 🤖 Life Assistant

> Seu assistente inteligente no Discord, com IA avançada, memória de contexto e respostas naturais.

---

## ✨ Funcionalidades

- 💬 Responde mensagens em servidores e DMs  
- 🧠 Usa IA (Groq + LLaMA 3) para respostas inteligentes  
- 🔁 Mantém contexto de conversas recentes  
- 📌 Responde mensagens antigas (com reply)  
- 😊 Reage automaticamente às mensagens  
- ⚡ Respostas rápidas e naturais  

---

## 🚀 Tecnologias utilizadas

- 🐍 Python  
- 💬 Discord.py  
- 🧠 Groq API (LLaMA 3)  
- 🌐 Flask (keep alive)
